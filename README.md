# word-counter
The 'Word Counter' is my web framework academic project created in Python using Django Framework

1. Install Python in your machine 

2. Install Django: Make sure you have Django installed in your Python environment. You can install it using pip:
   pip install django

3. Create a new Django project: Open Visual Studio Code and create a new Django project using the following command in the terminal:
   django-admin startproject word_counter_project

4. Define the views: Open the views.py file inside the word_counter app directory and define a view function to count words. Here's a simple example:
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////views.py in word_counter

from django.shortcuts import render

# defining function for wordcounter 
def counter(request): 
	# checking if method is POST or not 
	if request.method == 'POST': 

		# taking text input 
		text = request.POST['texttocount'] 

		# checking weather text is empty 
		# or not 
		if text != '': 

			# splitting the text and taking length 
			# of that 
			word = len(text.split()) 

			# defining boolean so that we can keep 
			# track of process later 
			i = True

			# returning HTML page with data, if calculated 
			# successfully 
			return render(request, 'counter.html', 
						{'word': word, 'text': text, 'i': i, 'on': 'active'}) 

		else: 
			# returning HTML page without data, if any 
			# error occurs 
			return render(request, 'counter.html', {'on': 'active'}) 

	else: 
		# returning HTML page if request.method is not POST 
		return render(request, 'counter.html', {'on': 'active'})

5. Define the URLs: Create a new file named urls.py inside the word_counter app directory and define the URL patterns for your views:
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////urls.py in word_counter

from django.urls import path
from . import views

urlpatterns = [
    path('', views.counter, name='counter'),  # Handle GET request
    path('counter/', views.counter, name='counter'),  # Handle both GET and POST requests
]

7. Include the app URLs in the project: Open the urls.py file inside the word_counter_project directory and include the URLs of the word_counter app:
////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////urls.py in word_counter_project

from django.contrib import admin
from django.urls import path
from word_counter import views  # Import views from the word_counter app

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', views.counter, name='counter'),  # Map the root URL to the counter view
]
